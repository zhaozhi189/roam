# Roam · 命名过程留底

> 起草日期:2026-05-12
> 用途:留下"为什么叫 Roam / 即境"的决策上下文,避免几个月后自己也忘了为啥选这个名字。

---

## 一、命名诉求

按 **"3D 扫描真实空间 → 浏览器 / 手机里漫游"** 这个核心场景做产品命名:

- 中英文都要好听、好读、好记
- 中文 2-3 字、有意境
- 英文短、好拼写、最好有双重含义
- 倾向"玩玩"的随性气质,不追求 SaaS 严肃感

## 二、8 个候选方案(4 种风格)

### 风格 A — 工具感 / SaaS 调

#### 1. 「拾间」/ Spatch

- **拾**(shí)= 拣起 / 收纳,**间**= 空间
- **Spatch** = Space + Snatch / Snap,新造词(域名很可能空)
- **气质**:像工程师做的工具,克制、专业
- 🎯 适合:做成 ProductHunt 上线那种风格

#### 2. 「拓影」/ Topia

- **拓**= 拓扑 / 拓展,**影**= 投影 / 影像
- **Topia** = Utopia 词根,带"理想空间"暗示
- **气质**:有理工科文艺范
- 🎯 适合:面向开发者 / 设计师

### 风格 B — 速度感 / 游戏调

#### 3. 「速间」/ Speedrun

- **速间** = 速通 + 空间;**Speedrun** 直接蹭那条爆款推文的原标签
- **气质**:Z 世代、游戏感、传播快
- 🎯 适合:面向年轻用户,病毒式增长

#### 4. 「即境」/ Roam ⭐ 最终选定

- **即境** = 即刻入境,**Roam** = 漫游
- **气质**:轻盈、即时、不需要解释
- 🎯 适合:大众消费产品

### 风格 C — 动作感 / 极简调

#### 5. 「步内」/ Stride

- **步**= 一步,**内**= 内部 / 里面;字面"踏入内部"
- **Stride** = 大步走;读音和"步内"在感觉上很对位
- **气质**:简洁、运动感
- 🎯 适合:个人项目代号、轻 App

#### 6. 「踏」/ Tread

- 一字一字双语,极简到底
- **气质**:像一个 Logo
- 🎯 适合:你只想要"一个 icon + 一个名字"的极简风

### 风格 D — 文艺感 / 情怀调

#### 7. 「在此」/ Hereto

- **在此** = 我在这里 / 此地;**Hereto** = 古英文 "to here / at this point"
- **气质**:文艺、有留白
- 🎯 适合:走情感路线("把家留下来""把回忆走一遍")

#### 8. 「行迹」/ Trace

- **行迹** = 走过的痕迹;**Trace** = 痕迹 / 描摹
- **气质**:有记录、归档、回忆感
- 🎯 适合:个人/家庭记忆类产品

## 三、AI 推荐过的两个

| 推荐 | 名字 | 推荐理由(初轮) |
|---|---|---|
| 🥇 综合首选 | 「拾间」/ Spatch | 原创度高、可注册、有故事可讲(拾起一个空间)、不撞名;商业可扩展性最高 |
| 🥈 玩玩首选 | 「速间」/ Speedrun | **自带传播种子**(蹭原 viral 推文标签);朋友群一发就 get |

## 四、最终选定 · 「即境」/ Roam

### 选定理由

- **轻盈、即时、不需要解释** —— 中英文都直接说出"漫游"这件事
- **Roam** 在英文里是个老词,读音两个音节就解决,任何文化背景都好念
- **「即境」** 中文有意境(即刻入境),不土不油
- 项目目录已建在 `/Users/zhi/Documents/dev/private/roam`
- 个人玩玩项目,不追求 SaaS 工具感,不追求 viral 病毒感,**轻盈即时最合**

### 没选其他的原因

| 候选 | 没选的原因 |
|---|---|
| 拾间 / Spatch | 太工具/商业,与"玩玩"的随性不符 |
| 拓影 / Topia | 受众太窄(理工科文艺范),圈层有限 |
| 速间 / Speedrun | 短期蹭热度可以,长期"Speedrun"会被游戏圈占走 |
| 步内 / Stride | Stride 太通用,Nike 已经把它带跑了 |
| 踏 / Tread | 太极简,在中文搜索 / 营销中辨识度低 |
| 在此 / Hereto | Hereto 略古老,大众不易感知 |
| 行迹 / Trace | Trace 撞名严重(GitHub / 数据库 / 监控产品都叫这个) |
