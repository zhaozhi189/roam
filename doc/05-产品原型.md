## 一、页面清单与导航(v0.2 重写)

MVP 必要页面 6 张(实际是 Android App 的页面 / 弹窗,非 Web SaaS 路由):

| # | 页面 | 形态 | 备注 |
|---|---|---|---|
| P1 | 落地页 | 仅推广,App 内不展示 | 介绍 Roam 是什么,引导下载 APK |
| P2 | 我的场景库 | App 启动默认页 | 场景卡片网格 + 导入按钮 |
| P3 | 导入场景 | 弹窗 / Bottom Sheet | 调系统 SAF 选 .spz 文件 |
| P4 | 场景详情 | App 内独立页 | 3D 预览 + 视频列表 + 分享 + 删除 |
| P5 | 本地漫游 | 全屏深色沉浸 | 核心体验;漫游 + 录制按钮 + 分享 |
| P6 | 帮助文档 | App 内 / 网页都有 | 5 步指南:装 Roam → Scaniverse 扫 → 导入 → 漫游录视频 → 微信分享 |

**v0.2 关键变化**:
- ❌ 删除「公开 URL」「登录」「分享链接」概念(ADR-004 本地化)
- ✅ 新增「录制视频」「分享到微信」「场景库本地」概念
- P5 形态收敛为单一(深色沉浸)— 不再有「桌面版」(MVP 是 Android App)

**首版不做**:个人设置页、社交分享卡、收藏夹、评论、场景列表分页、多语言。

**导航关系**(简化):

```
首次启动 → P1 落地页(可选,引导用户)→ P2 场景库

P2 场景库 ──[+ 导入]──> P3 导入弹窗 ──完成──> P4 场景详情
            ──[点卡片]──> P4 场景详情
            ──[? 帮助]──> P6 帮助

P4 场景详情 ──[漫游]──> P5 本地漫游
            ──[录制]──> P5 + 录制中状态
            ──[分享]──> Android Share Intent → 系统面板 → 微信

P5 本地漫游 ──[底部 录制按钮]──> 录制中状态
            ──[底部 分享按钮]──> Android Share Intent
            ──[← 返回]──> P4
```

## 四、P3 导入场景(Bottom Sheet)

Android 调用 Storage Access Framework 选 `.spz` 文件,复制到 App internal storage。

```
┌─────────────────────────────────────────────┐
│  导入新场景                              [×]│
├─────────────────────────────────────────────┤
│   ┌─────────────────────────────────────┐   │
│   │  选择 .spz 文件                     │   │
│   │       [打开文件选择器]               │   │
│   │  单文件 ≤ 30 MB · Scaniverse 直出   │   │
│   └─────────────────────────────────────┘   │
│   场景名称: [_____________________________] │
│              [取消]    [开始导入]            │
└─────────────────────────────────────────────┘
```

**导入中**(从外部存储复制到 App private storage):

```
┌─────────────────────────────────────────────┐
│  正在导入:卧室                              │
│  ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░  62%   7.4MB / 12MB    │
│   [取消导入]                                 │
└─────────────────────────────────────────────┘
```

**操作要确认**:

| 场景 | 行为 |
|---|---|
| 文件 > 30MB | 前端拒绝,提示「文件超过 30MB → P6 帮助文档」 |
| 文件类型错(非 .spz) | 拒绝,提示用 Scaniverse 重新导出 SPZ |
| 场景名为空 | 用文件名(去扩展名)兜底 |
| 导入中关闭 Sheet | **MVP 简化:关闭即取消**(不做后台导入) |
| 导入失败 | 红色提示 + 重试按钮 |
| 导入成功 | Sheet 关闭 + Toast「场景已导入」+ 自动跳 P4 |

## 五、P4 场景详情

```
┌─────────────────────────────────────────────────┐
│  ← 返回                                          │
│  卧室 [✏️]                                       │
│  创建于 2026-05-12 · 12 MB · .spz                │
│                                                  │
│  ┌────────────────────────────────────────────┐ │
│  │       [ 3D 预览 · 站位 ]                    │ │
│  │   M1 之后接 PlayCanvas + @spz-loader        │ │
│  └────────────────────────────────────────────┘ │
│  虚拟摇杆移动 · 拖拽视角 (MVP 无碰撞)            │
│  [进入漫游 →]                                    │
│                                                  │
│  ─────────── 录视频 + 分享 ───────────           │
│  [🔴 录制漫游视频(60s)]  [🔗 分享最近视频]      │
│                                                  │
│  已录制视频(本地):                              │
│   ▶ 卧室漫游_001.mp4 · 60s · 18MB  [分享微信→]  │
│   ▶ 卧室漫游_002.mp4 · 45s · 14MB  [分享微信→]  │
│                                                  │
│  ─────────── 危险操作 ───────────                │
│   [🗑 删除场景]                                  │
└─────────────────────────────────────────────────┘
```

**操作要确认**:

| 操作 | 行为 |
|---|---|
| ✏️ 改名 | inline 编辑 → 回车保存 |
| 进入漫游 | 跳 P5 全屏沉浸模式 |
| 🔴 录制 | 跳 P5 + ?record=1,自动启动 MediaProjection |
| 🔗 分享最近视频 | 调 Android Share Intent,弹系统面板,选微信 |
| 视频列表项 | 点 ▶ 预览;点「分享微信→」直接分享该视频 |
| 删除场景 | 二次确认对话框,需输入场景名才能删;一并清除该场景所有录制视频 |
| 视频列表条数 | MVP 不限,长按可删(类似 Photos App) |

## 六、P5 本地漫游(核心体验,深色沉浸)

```
┌────────────────────────────────────────┐
│ ← 卧室                  ⋮              │ ← 顶部 glass bar
│   本地场景 · 12 MB · .spz              │
├────────────────────────────────────────┤
│                                         │
│  [@spz-loader 加载 12 MB · 62%]        │ ← 加载中(可隐藏)
│                                         │
│       [ 3D 漫游画布 · 占满 ]            │
│                                         │
│            ╭───╮                       │ ← 虚拟摇杆(底部居中)
│            │ ⊙ │                       │
│            ╰───╯                       │
│                                         │
│  [全屏]  [🔴 录制视频]  [🔗 分享]       │ ← 底部 action bar
└────────────────────────────────────────┘

录制中状态(?record=1):
┌────────────────────────────────────────┐
│ ← 卧室                  ⋮              │
│   ┌────────────────────┐               │
│   │ ● REC  00:23 / 01:00│              │ ← 红色 REC 状态条
│   └────────────────────┘               │
│         [ 3D 漫游画布 ]                 │
│            ╭───╮                       │
│            │ ⊙ │                       │
│            ╰───╯                       │
│  [全屏]  [⏹ 停止]  [🔗]                │
└────────────────────────────────────────┘
```

**操作要确认**:

| 项 | 行为 |
|---|---|
| 漫游控制 | 虚拟摇杆 + 单指拖视角(MVP 无物理碰撞,可穿墙) |
| 加载中 | glass 进度条 + 「@spz-loader 加载 X MB」 |
| 全屏 | 隐藏 / 显示顶部 + 底部 bar(类似 YouTube) |
| 录制 | 点 🔴 → 弹 MediaProjection 系统权限 → 允许 → 开始录制 |
| 录制时长 | 默认上限 60s(微信 25MB 限制),时间到自动停 |
| 录制中再点录制按钮 | 改为 ⏹ 停止 → 视频保存到本地,Toast「视频已保存到本地」 |
| 分享 | 调 Android Share Intent,弹系统面板,选微信 |
| ⋮ 菜单 | 关于此场景(大小、创建时间)/ 删除场景 |
| **3D 交互细节** | 摇杆灵敏度 / 移动速度 / 视角加速度 等 → M1 跑通 demo 后定 |

## 八、不做什么(MVP 边界 v0.2)

- ❌ 用户登录系统(本地 App 无身份)
- ❌ 公开链接 / 公网分享(Phase 2 退路)
- ❌ 场景隐私(本地全私有,不需要选项)
- ❌ 评论 / 点赞 / 收藏(单机 App 无社交)
- ❌ iOS / Web 版(ADR-005 推迟)
- ❌ 微信 OpenSDK 深度集成(Share Intent 够用)
- ❌ 物理碰撞(MVP 无,Q8 (a))
- ❌ 自动运镜路径(F-09 Phase 2)
- ❌ 多语言 / 付费 / 文件夹标签
- ❌ 强制水印(画面内贴牌)
