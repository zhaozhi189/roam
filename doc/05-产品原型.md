## 一、页面清单与导航

MVP 必要页面 6 张,首版不做的另列。

| # | 页面 | 路径(占位) | 登录要求 | 备注 |
|---|---|---|---|---|
| P1 | 落地页 | `/` | 否 | 未登录默认页 |
| P2 | 我的场景 | `/dashboard` | 是 | 登录后默认页 |
| P3 | 上传场景 | 弹窗,非独立路由 | 是 | 在 P2 上叠加 |
| P4 | 场景详情 | `/scene/:id` | 是(仅本人) | 改名/删除/复制公开 URL |
| P5 | 公开浏览 | `/s/:shortId` | 否 | 3D 漫游,核心体验 |
| P6 | 帮助文档 | `/help` | 否 | 静态:扫描指南 + SplatTransform 教程 |

**首版不做**:个人设置页、社交分享卡片预览页、收藏夹、评论、场景列表分页。

**导航关系**:

```
未登录:                          登录后:
P1 落地页 ──[GitHub 登录]──>  P2 我的场景
P5 公开浏览(分享链直达)         ├──[点卡片]──>     P4 场景详情
P6 帮助(独立访问)               ├──[点 + 上传]──>  P3 上传弹窗 ──完成──> P4
                                └──[头像菜单]──>   登出
```

**操作要确认**:

| 操作 | 行为 | 备注 |

## 四、P3 上传场景(弹窗)

```
┌─────────────────────────────────────────────┐
│  上传新场景                              [×]│
├─────────────────────────────────────────────┤
│                                              │
│   ┌─────────────────────────────────────┐   │
│   │                                     │   │
│   │  拖拽 .sog 或 .compressed.ply 文件  │   │
│   │       或 [点击选择文件]              │   │
│   │                                     │   │
│   │  单文件不超过 30 MB                  │   │
│   │  不接受原始 .ply,见 P6 教程         │   │
│   └─────────────────────────────────────┘   │
│                                              │
│   场景名称: [_____________________________] │
│   (默认:文件名,可改)                       │
│                                              │
│              [取消]    [开始上传]            │
└─────────────────────────────────────────────┘
```

**上传中**:

```
┌─────────────────────────────────────────────┐
│  正在上传:卧室                              │
│  ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░  62%   7.4MB / 12MB    │
│                                              │
│   [取消上传]                                 │
└─────────────────────────────────────────────┘
```

**操作要确认**:

| 场景 | 行为 |
|---|---|
| 文件 > 30MB | 前端直接拒绝,提示「文件超过 30MB → P6 压缩教程」 |
| 文件类型错 | 拒绝,提示支持的扩展名 |
| 场景名为空 | 用文件名(去扩展名)兜底 |
| 上传中关闭弹窗 | **MVP 简化:关闭即取消**(不做后台上传) |
| 上传失败 | 红色提示 + 重试按钮 |
| 上传成功 | 弹窗关闭 + Toast「场景已创建」+ 自动跳 P4 |

## 八、不做什么(MVP 边界)

- ❌ 用户设置页(头像/简介都从 GitHub 拉)
- ❌ 场景隐私(私有 / 密码访问)
- ❌ 评论 / 点赞 / 收藏
- ❌ 封面图自定义上传(只用自动截图)
- ❌ 社交分享卡片预览页(交给 OG tag 让平台自渲染)
- ❌ 多语言(MVP 只中文)
- ❌ 付费 / 订阅
- ❌ 场景文件夹 / 标签
- ❌ 强制水印(底部 logo 而非画面内贴牌)
