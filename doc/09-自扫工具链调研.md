# 自扫 3DGS 工具链调研(2026-05-17)

> 缘起:Scaniverse(原计划)在国内 Niantic 登录受阻;为 M3+ 找替代方案。
> 关联:[06 v1.2](06-技术可行性验证.md)、[ADR-004 本地优先](decisions/ADR-004-MVP-方向调整-本地优先.md)

## 目标

找 1+ 工具能在 zhi 手机/桌面跑通:**真实空间 → 3DGS .ply/.sog/.spz 输出 → 放到 Roam App 内 SAF 选**。

理想:Android App,完全本地处理,免费,国内可访问。

## 候选方案

### A · Scaniverse(Niantic)— 当前选型,卡

| 项 | 状态 |
|---|---|
| 平台 | Android + iOS |
| 输出 | SPZ(Niantic 自研,Roam 已支持) |
| 处理 | on-device |
| 国内 | ⚠️ 登录卡(Niantic 账号 / Google 登录受限) |
| 免费 | ✅ |
| zhi 实测 | 装上但登不上 |

**绕过尝试**:
- Apple ID 登录(国内能用)
- 挂代理首登一次(本地处理,后续断网可用)
- 改 hosts / DNS

### B · KIRI Engine

| 项 | 状态 |
|---|---|
| 平台 | Android + iOS + Web |
| 输出 | 3DGS(.ply 等) |
| 处理 | **云端**(免费版限次,Pro 收费) |
| 国内 | ✅ 可访问 |
| 免费 | 受限 |
| zhi 实测 | 之前装了,「定位不准、不能正常扫描」 |

**重试方向**:
- 关 LiDAR 依赖,用 Featureless Object Capture 模式
- 选「Photogrammetry」而非「3DGS」(后者云端 quota 紧张)

### C · Polycam

| 项 | 状态 |
|---|---|
| 平台 | iOS / Web(Android 受限) |
| 输出 | mesh + Splat(Web 版) |
| 处理 | 云端 |
| 国内 | Web 可访问;App store 国区有 |
| 免费 | 受限,带水印 |

**问题**:zhi 是 Android,Polycam Android 没 Splat 输出。可用 Web 版上传照片/视频。

### D · Postshot(桌面)

| 项 | 状态 |
|---|---|
| 平台 | Windows(支持 Linux WSL) |
| 输出 | 3DGS .ply / .ksplat |
| 处理 | 本地 NVIDIA GPU(需 RTX) |
| 国内 | ✅ |
| 免费 | ✅ |

**流程**:手机拍视频 → 传桌面 → Postshot 训练 30 分钟-几小时 → 导出 .ply → 传回手机 SAF → Roam 加载

**痛点**:需要 NVIDIA GPU + 桌面环节,违反「移动端闭环」目标

### E · gsplat.studio(在线 SaaS)

| 项 | 状态 |
|---|---|
| 平台 | Web |
| 输出 | .ply / .splat |
| 处理 | 云端 |
| 国内 | ⚠️ 需测试 |
| 免费 | 限次 |

**流程**:手机拍视频上传 → 等几分钟训练 → 下载

### F · Luma AI

| 项 | 状态 |
|---|---|
| 平台 | iOS + Web |
| 输出 | NeRF + 3DGS |
| 国内 | ⚠️ Web 可访问,App 国区下架 |
| 免费 | 限次 |

Android 不支持,zhi 不直接可用。

### G · RealityScan(Epic Games)

| 项 | 状态 |
|---|---|
| 平台 | iOS + Android |
| 输出 | **Mesh**(非 3DGS) |
| 国内 | 需 Epic 账号 |

**不符合**:输出 mesh 不是 3DGS,Roam 不支持加载。

### H · 3D Scanner App(iOS)

仅 iOS,跳过。

### I · 自训练桌面(Nerfstudio / Gaussian Splatting 原版)

技术可行,需要:
- NVIDIA GPU(RTX 3060+ 推荐)
- Linux / WSL + CUDA + Python
- COLMAP 算 camera pose
- 训练 30 分钟-几小时

zhi 桌面是 Mac(M 系列芯片),Nerfstudio 支持 MPS 但慢且不稳。**不推荐**。

## 推荐方案

按 zhi 「Android only + 本地优先 + 国内可用 + 免费」严格条件,**完全满足**的方案近乎没有。

务实排序:
1. **Scaniverse 绕过登录**(挂代理首登,之后离线用)— 最贴合原计划
2. **KIRI Engine 重试**(关 LiDAR,用 Featureless Capture)— 国内可访问 + 移动端
3. **Polycam Web + 用电脑过一道**(Web 端国内能用,但流程多)
4. **桌面 Postshot**(zhi Mac 不行,跳)→ 借朋友 PC 偶尔训练

## 行动计划

### 短期(M3)
- zhi 试 Scaniverse 用 Apple ID 登录 / VPN 绕一次
- 如果不行,KIRI Engine 重试,用最低期望(扫桌上水杯 / 玩偶)

### 中期(M4+)
- 找个 PC 朋友帮训 几个 demo 场景(Postshot,zhi 拍视频)
- 把成果 vendor 进 App 作 demo 场景(scenes/zhi-room.ply 等)

### 长期(M5+)
- 关注 Niantic 是否开放 Scaniverse 国内访问
- 关注 Polycam Android Splat 支持
- 关注 mobile 端原生 3DGS 训练库(目前 Magic7 Pro 算力够,缺成熟 App)

## 不在调研范围

- iOS only 工具(zhi Android 主测)
- 闭源企业方案(Lightcraft / OpenSplat 等)
- 自训练桌面(Mac 不友好,需 NVIDIA)
