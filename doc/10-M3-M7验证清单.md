```bash
# 装最新
adb install -r /Users/zhi/Documents/dev/private/roam/client/app/build/outputs/apk/debug/app-debug.apk

# 跑完整 e2e(8 段)
./scripts/e2e-test.sh

# 单跑某场景
adb shell am broadcast -a com.roam.app.AUTO --es cmd apartment
adb shell am broadcast -a com.roam.app.AUTO --es cmd spz

# Deep link
adb shell am start -W -a android.intent.action.VIEW -d "roam://scene/guitar"

# 清 App data(localStorage / 录屏 / 用户场景全部清空)
adb shell pm clear com.roam.app
```

## 已修复的 bug(本次会话)

| Bug | 现象 | 修法 |
|---|---|---|
| Cannot access 'pcApp' before init | inline script TDZ 中断,所有按钮 disabled | userSettings / pcApp / currentCam 提到顶部 init |
| Java bridge non-injected object | 切 indoor 模式 toast 报错 | 不 cache `window.RoamBridge.toast`,直接调用 |
| SPZ 蜥蜴 splat 出视野 | 切 1.5x 距离后看不到 | radius 回 1.8x(v2 已知 work) |
| Indoor 模式 + 物体场景看不到 | 切 indoor 站物体内部全黑 | 切 indoor 时 toast 警告 + 引导切回 |
| GSplatResource.instantiate undef | SPZ 真渲染挂 | monkey patch instantiate,asset 包路径 |
| 沉浸不真全屏 | .card backdrop-filter stacking context | detach canvas-wrap 到 body |
| 录视频微信不能播 | WebM 不兼容 | WebCodecs MP4 输出 |
