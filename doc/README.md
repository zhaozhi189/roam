## 一句话

Roam(中文「即境」)把"用手机扫房间,在浏览器里漫游"这件事做成**人人都能用的工具 + 可分享的 URL**,核心技术栈是 3D Gaussian Splatting + PlayCanvas + SplatTransform 2.0,**不依赖大模型**。
