如何制作可漫游的 3D 场景

从手机扫描到微信视频分享,全程在手机上完成,大约 30 分钟。本文档随 MVP 持续更新。

Roam 是什么

Roam 是一款 Android 单机 App,把你扫描的真实空间(卧室、咖啡馆、展览等)变成可在手机里漫游的 3D 场景,**录视频发微信**给朋友看。

核心约束:

  • 仅支持 Android(MVP 阶段不做 iOS)
  • 仅支持 .spz 文件(Niantic Scaniverse 直出格式,~10MB / 场景)
  • 本地存储,不上传任何云,不需要登录
  • 分享是录视频微信发,接收方看视频不漫游(Phase 2 考虑公网链接)

1装 Roam APK

从 GitHub Release 或 zhi 直接发你的 APK 下载到 Magic7 Pro 或同代 Android 旗舰(骁龙 8 Gen 系列 / 天玑 9000 系列)。

  • 系统要求:Android 10+,WebView 系统版本 ≥ 100
  • 装上后会提示「未知来源安装」,选「允许」
  • 首次启动会请求存储权限(读 .spz 文件)和录屏权限(录视频)

⚠️ 中端机 / 千元机 MVP 不保证流畅,见 [03 § 六 性能关键点](../doc/03-技术架构.md)

2扫描房间

推荐 Scaniverse(Niantic 出品,免费,Android 原生支持):

  • Play Store 在国内不可用 → 用 APKPure / Uptodown 镜像下 APK
  • App 内选 Splat 模式(不要选 Mesh)
  • 绕着空间走一圈拍照,设备端 60-90 秒处理出 splat 场景
  • 导出格式选 SPZ(Niantic 开源,~10MB,Roam 唯一支持的格式)

扫描技巧

  • 光照均匀:避免逆光、强阴影
  • 慢速移动:每秒移动 ≤ 30 cm
  • 多角度:每个物体至少绕 180°
  • 纹理丰富优先,纯白墙 / 玻璃尽量少

3导入到 Roam

Scaniverse 导出的 .spz 文件默认在 /sdcard/Download/ 或 Scaniverse Exports 目录。

  1. 打开 Roam → 点右上角 + 导入场景
  2. 系统弹出文件选择器(Storage Access Framework)
  3. 选你的 .spz 文件 → 输入场景名 → 确认
  4. Roam 把文件复制到 App 私有存储(几秒钟),场景出现在场景库

支持的格式:.spz 单一格式。不支持原始 .ply(太大,几百 MB)、.sog(PlayCanvas 自家格式,M0 评估后不再使用)。

4漫游 + 录视频

点开任意一个场景:

  • WebView + PlayCanvas + @spz-loader/playcanvas 加载 SPZ,~3 秒可漫游
  • 左下角 虚拟摇杆 控制走动,右下区拖拽控制视角
  • MVP 无物理碰撞(可穿墙) — 接收方看视频不感知,你自己别撞
  • 找到喜欢的运镜路径,点底部 🔴 录制视频 → 系统弹「即将开始捕获屏幕」对话框 → 允许
  • 录制中顶部有红色 REC 状态条 + 倒计时(默认 60s 上限,适配微信)
  • 录完点停止 → MP4 自动保存到本地视频列表

5微信分享

录完视频后:

  1. 在场景详情页或漫游页底部点 分享
  2. Android 弹系统分享面板,列出已装的所有支持视频分享的 App
  3. 选 微信 → 选「发送给朋友」/「分享到朋友圈」
  4. 朋友收到一段 60s 流畅运镜视频,**看着像 3D vlog 短视频**

视频规格:1080p@30fps H.264 MP4,60 秒,文件 ≤ 25MB(微信最大限制)。

常见问题

为什么不接受原始 .ply 或 PlayCanvas 的 .sog?

Roam MVP 用 Niantic SPZ 格式:Scaniverse 直接导出,@spz-loader/playcanvas runtime 直接加载,**全程在手机闭环不需要离开 App**。原 .sog 路径需要电脑跑 SplatTransform 处理,体验断点(参见 doc/03 v0.3 Q7 关闭)。

为什么没有公开链接?接收方能自己漫游吗?

MVP 是纯本地版,没有云端,所以没有公开链接。接收方在微信看到的是视频,不能自己漫游。Phase 2 可能接入 Cloudflare 做公网版,让朋友扫码也能进 Roam 漫游。

为什么不支持 iOS?

个人项目控制范围。iOS 在 ADR-005 明确推迟。后续如果反响好,Phase 2 用 Capacitor / Tauri Mobile 做跨端。

为什么不做云端训练 / 自动一键扫描?

MVP 阶段没有 GPU 预算,也不接 Polycam / Luma API(WorkOS 在国内还有访问问题)。Scaniverse 设备端 60-90 秒处理就够日常用。

漫游可以穿墙吗?

MVP 是的,SPZ 不带碰撞体,Roam 暂不计算软碰撞。**你自己别撞墙就行**,录视频时按设计运镜路线走。Phase 2 视情况上 PlayCanvas raycaster 软碰撞(参见 Q8 决策)。
